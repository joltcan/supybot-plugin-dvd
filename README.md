supybot-plugin-dvd
==================

DVD trivia plugin with sqlite3.

Installation
------------
Put it in your plugins folder and do a !load DVD.

Usage
-----
!dvd
    gives you a random DVD
!add <title of movie>
    Adds a movie
!delete <title of movie>
    Deletes a movie
!stats
    Shows you how many movies there are.
